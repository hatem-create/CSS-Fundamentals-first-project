# Product Preview Card Component

A clean and simple web page built to practice the absolute fundamentals of HTML5 and CSS3. This project focuses on document structuring, basic styling, and typography without relying on complex layouts or responsive frameworks.

## 🛠️ Features & Concepts Practiced
* **HTML5 Structure:** Using semantic tags (`<main>`, `<article>`, `<div>`) correctly.
* **Typography:** Importing and applying custom Google Fonts (Fraunces and Montserrat).
* **CSS Fundamentals:** 
  * Working with selectors and class naming.
  * Basic box model properties (`padding`, `margin`, `border-radius`).
  * Color management using CSS variables or hex codes.
  * Image handling within containers.

## 📁 Project Structure
* `index.html` - The main structure of the card component.
* `styles.css` - Custom styling for fonts, colors, and layout elements.
* `/images` - Asset folder containing the project images.

## 📝 What I Learned
During this project, I focused on learning how to style text elements cleanly and ensure elements align correctly using basic CSS properties before moving on to advanced layouts like Flexbox or Grid.
